## Installation

Install

```sh
npm install @erayjs/milyui
```

For CDN

```sh
<script src="https://unpkg.com/@erayjs/milyui"></script>

<script src="https://cdn.jsdelivr.net/gh/erayjsx/milyui@0.0.3/dist/bundle.js"></script>
```
